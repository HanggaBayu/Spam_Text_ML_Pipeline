| Dataset | [nama dataset]([https://www.kaggle.com/](https://www.kaggle.com/datasets/team-ai/spam-text-message-classification)) |
| Masalah | Pesan spam adalah pesan yang tidak diinginkan yang dapat mengganggu pengalaman pengguna, menyebabkan potensi ancaman keamanan, dan membanjiri kotak masuk email atau platform komunikasi lainnya. Peningkatan kecerdasan buatan telah memberikan peluang baru bagi penjahat dunia maya untuk membuat pesan spam yang lebih sulit dideteksi, seperti menggunakan teknik obfuskasi teks, citra, atau tautan terenkripsi. Oleh karena itu, pengembangan sistem deteksi pesan spam yang efektif perlu terus-menerus beradaptasi dengan inovasi-inovasi ini untuk menjaga integritas komunikasi digital dan melindungi pengguna dari potensi ancaman dan gangguan. |
| Solusi machine learning | Solusi machine learning untuk pemisahan atau klasifikasi pesan spam melibatkan pemanfaatan algoritma pembelajaran mesin untuk secara otomatis mengenali dan membedakan pesan-pesan yang dianggap spam dari yang tidak. Dalam konteks ini, teknologi tersebut memerlukan pengumpulan data latihan yang mencakup contoh-contoh pesan spam dan non-spam |
| Metode pengolahan | Pembuatan model machine learning menggunakan TensorFlow untuk melakukan klasifikasi biner pada pesan. Proses dimulai dengan pengolahan data, di mana teks pesan diubah menjadi vektor numerik menggunakan TextVectorization. Model dibangun dengan menggunakan lapisan-lapisan seperti embedding, pooling, dan dense layers. Data kemudian dibagi menjadi set pelatihan dan validasi. Model dikompilasi dengan fungsi kerugian binary crossentropy dan optimizer Adam, serta dilatih menggunakan dataset yang telah diolah. Model ini dirancang untuk mengklasifikasikan pesan menjadi dua kategori biner 1(ham)  atau 0 (spam) berdasarkan isi "message", dan evaluasi kinerjanya menggunakan metrik seperti binary accuracy. Keseluruhan, proses pipeline mencakup data ingestion, data validation, data preprocessing, training, dan evaluation |
| Arsitektur model | Arsitektur model ini menggunakan lapisan Text Vectorization untuk mengonversi teks pesan, diikuti oleh lapisan Embedding untuk representasi vektor numerik. Global Average Pooling Layer digunakan untuk merata-ratakan vektor, dan beberapa lapisan dense dengan aktivasi relu digunakan untuk pemrosesan fitur. Lapisan output menggunakan sigmoid untuk menghasilkan output klasifikasi biner. Secara keseluruhan, model ini dirancang untuk mengambil teks sebagai input dan menghasilkan prediksi apakah pesan tersebut spam atau bukan |
| Metrik evaluasi | Metric yang diterapkan pada model melibatkan BinaryAccuracy, TruePositive, FalsePositive, TrueNegative, dan FalseNegative untuk mengevaluasi kinerja model dalam melakukan klasifikasi. |
| Performa model | Model mencapai Binary Accuracy sebesar 97.70%, menunjukkan akurasi yang tinggi. AUC mencapai 97.38%, menandakan kemampuan model membedakan kelas. Loss sekitar 0.146, menunjukkan model mendekati distribusi yang benar. False Positives dan True Positives masing-masing 25 dan 970, sedangkan False Negatives dan True Negatives masing-masing 1 dan 133. Model cenderung memberikan prediksi positif yang akurat. |
